## bson4s

### Usage

This library is only available for Scala 3.3.3 (lts version) and later.

To use the latest version, include the following in your `build.sbt`:

```scala
libraryDependencies ++= Seq(
  "se.thanh" %% "bson4s" % "@VERSION@"
)
```
